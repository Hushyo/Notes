# 上传与拉取

## 拉取

> 将本地与远程同步

- cd e:/git 打开本地仓库
- git pull
- git clone 仓库地址
  克隆仓库到本地

## 上传

1. 全部更新: git add -A
   选择更新: git add 文件名
2. git commit -m "备注"
3. git push

> git add 文件夹/文件名 可以直接把整个文件夹+文件传上去

## 修改

- git mv **<old\> <new\>** 文件改名
- git rm <file\> 删除文件
  git rm -r <file\> 删除文件夹
